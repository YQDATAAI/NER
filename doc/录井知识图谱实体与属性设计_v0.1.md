# 录井知识图谱实体与属性设计（Mud Logging Knowledge Graph Schema）

版本：v0.1\
生成时间：2026-02-28 05:44:47

------------------------------------------------------------------------

# 一、总体设计原则

录井图谱属于**事件驱动型知识图谱（Event-Driven Knowledge
Graph）**，核心围绕：

-   井（Well）
-   钻井过程（Drilling Operation）
-   录井事件（Logging Events）
-   样品与观察（Sample & Observation）
-   油气显示（Hydrocarbon Show）
-   证据与断言（Evidence & Assertion）

图谱结构应支持：

-   时间 + 深度双维度建模
-   事件序列建模
-   样品级证据绑定
-   多来源冲突管理
-   可追溯性（Evidence-bound）

------------------------------------------------------------------------

# 二、核心实体列表

## 1️⃣ Well（井）

  - 属性               类型       说明
  ------------------ ---------- ------------
  - well_id            string     井唯一ID
  well_name          string     井名
  block              string     区块
  field              string     油气田
  surface_location   geo        地理坐标
  kb_elevation       float      转盘面高程
  spud_date          datetime   开钻日期
  completion_date    datetime   完钻日期

------------------------------------------------------------------------

## 2️⃣ Wellbore（井眼）

  - 属性              类型     说明
  ----------------- -------- ------------------------
  - wellbore_id       string   井眼ID
  well_id           string   所属井
  type              enum     直井 / 定向井 / 水平井
  md_total_depth    float    井深（MD）
  tvd_total_depth   float    垂深（TVD）
  kickoff_depth     float    造斜点

------------------------------------------------------------------------

## 3️⃣ DrillingOperation（钻井过程）

  - 属性                  类型       说明
  --------------------- ---------- ------------
  - operation_id          string     过程ID
  wellbore_id           string     所属井眼
  drilling_crew         string     钻井小队
  drilling_fluid_type   string     钻井液类型 default 水基
  start_time            datetime   开始时间
  end_time              datetime   结束时间

------------------------------------------------------------------------

## 4️⃣ DrillingEvent（钻井事件）

 -  属性            类型       说明
  --------------- ---------- -----------------------
  - event_id        string     事件ID
  operation_id    string     所属过程
  timestamp       datetime   时间
  depth_md        float      当前井深
  event_type      enum       起钻/下钻/接单根/异常
  rop             float      机械钻速
  wob             float      钻压
  rpm             float      转速
  torque          float      扭矩
  pump_pressure   float      泵压

------------------------------------------------------------------------

## 5️⃣ Sample（岩屑样品）

  - 属性                  类型       说明
  --------------------- ---------- ----------
  - sample_id             string     样品ID
  wellbore_id           string     所属井眼
  depth_md              float      采样深度
  sample_time           datetime   采样时间
  lag_time              float      迟到时间
  lithology_primary     string     主岩性
  lithology_secondary   string     次岩性
  color                 string     颜色
  texture               string     结构
  sorting               string     分选
  roundness             string     圆度

------------------------------------------------------------------------

## 6️⃣ LithologyObservation（岩性观察）

  - 属性             类型     说明
  ---------------- -------- ----------
  - observation_id   string   观察ID
  sample_id        string   所属样品
  description      text     描述文本
  confidence       float    置信度
  interpreter      string   解释人员

------------------------------------------------------------------------

## 7️⃣ GasShow（气测显示）

  - 属性          类型       说明
  ------------- ---------- ----------------------
  - gas_id        string     记录ID
  wellbore_id   string     所属井眼
  depth_md      float      深度
  timestamp     datetime   时间
  total_gas     float      全烃含量
  methane       float      CH4
  ethane        float      C2
  propane       float      C3
  butane        float      C4
  heavy_ratio   float      重烃比
  gas_type      enum       背景气/异常气/峰值气

------------------------------------------------------------------------

## 9️⃣ DepthInterval（深度区间）

  - 属性          类型     说明
  ------------- -------- ----------
  - interval_id   string   区间ID
  wellbore_id   string   所属井眼
  md_start      float    起始深度
  md_end        float    终止深度
  tvd_start     float    起始垂深
  tvd_end       float    终止垂深
  strat_unit    string   层位名称

------------------------------------------------------------------------

## 🔟 Interpretation（解释结论）

  - 属性                               类型       说明
  -------------------               ---------- --------------------
  - interpretation_id                string     结论ID
  Interpretation_type                enum       岩性/储层/含油气性
  Interpretation_value               string     结论值
  Interpretation_method              string     解释方法
  Interpretation_confidence          float      置信度
  Interpretation_created_at          datetime   生成时间

------------------------------------------------------------------------

## 11️⃣ Evidence（证据）

  - 属性                类型       说明
  ------------------- ---------- ---------------------
  - evidence_id         string     证据ID
  source_type         enum       文本/样品/气测/仪器
  source_ref          string     来源定位信息
  extractor_version   string     抽取器版本
  confidence          float      置信度
  created_at          datetime   时间

------------------------------------------------------------------------

# 三、关系（Edges）建议

-   Well → HAS_WELLBORE → Wellbore
-   Wellbore → HAS_OPERATION → DrillingOperation
-   DrillingOperation → HAS_EVENT → DrillingEvent
-   Wellbore → HAS_SAMPLE → Sample
-   Sample → HAS_OBSERVATION → LithologyObservation
-   Wellbore → HAS_GASSHOW → GasShow
-   Wellbore → HAS_OILSHOW → OilShow
-   Interpretation → ABOUT_INTERVAL → DepthInterval
-   Interpretation → SUPPORTED_BY → Evidence
